Frontend-разработчик с более чем 4 годами опыта полного цикла: от проектирования архитектуры с нуля до вывода продукта в продакшен. Обладаю опытом руководства командой: формировал отдел разработки, проводил собеседования и менторинг junior-специалистов, отвечал за качество кода через code review и координацию фронтенд-части релизов.

Технический стек:
- Базовые технологии: HTML5, CSS3, JavaScript (ES6+)
- Основной стек: Vue.js, TypeScript, Nuxt, Quasar
- Инструменты: Vite, Webpack, SASS/SCSS, Docker, Vitest

Постоянно развиваю экспертизу в современных подходах к разработке на Vue.js-экосистеме, уделяя внимание производительности, масштабируемости и поддержанию кодовой базы.
